7. POSTMAN KOLEKCIJA SETUP
===========================================

1. Kreiraj novu kolekciju "Chat API"
2. Dodaj environment sa varijablama:
   - BASE_URL: http://localhost:8000/api
   - TOKEN: (prazno, popuniće se nakon login-a)

3. Dodaj pre-request script za automatsko postavljanje tokena:
   if (pm.response.code === 200) {
       const response = pm.response.json();
       if (response.data && response.data.token) {
           pm.environment.set("TOKEN", response.data.token);
       }
   }

4. Dodaj test script za validaciju:
   pm.test("Status code is 200", function () {
       pm.response.to.have.status(200);
   });
   
   pm.test("Response has success field", function () {
       const response = pm.response.json();
       pm.expect(response).to.have.property('success');
   });
